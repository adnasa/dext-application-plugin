# dext-application-plugin

## !!Work in Progress!!

> Dext plugin to search local applications

## Install

```bash
$ dpm install dext-application-plugin
```

*Requires [Dext](https://github.com/vutran/dext).*

## Usage

In Dext, type `app`, then your query.
